## requirements 🙀

- cargo 🦀
   install with 
 ```
$ curl https://sh.rustup.rs -sSf | sh
 ```
 
## installation ❤️‍🩹

```
$ git clone --depth=1 https://github.com/BinaryBrainiacs/yarsi
$ cd yarsi 
$ cargo install --path .
```
